# Algo Workspace

C++ and Python algorithm competition environment with devcontainer.

## Quick Start

Open in VS Code with Dev Containers extension.

- `Ctrl+Shift+B` — build current C++ file
- `F5` — build & debug
- `Ctrl+Shift+P` → *Tasks: Run Task* → *run active file* — build & run

## Tools

- **C++**: g++ (C++17), GDB, clangd, clang-format
- **Python**: Python 3, Ruff, Ty
